# Yeast-Display-Demo
Yeast Display Antibody Discovery Demo Dataset

This dataset demonstrates a typical yeast surface display campaign workflow:

Round 1 – Initial binder population detection
Round 2 – Enrichment
Round 3 – Stringency selection
Round 4 – Off-rate / low concentration selection

The dataset includes:
- Raw FCS files
- Gating strategy
- Example clone binding curves
